Painting the Web

CSS provides powerful tools to manipulate color and imagery. Understanding how to use these effectively is the difference between a plain site and a professional one.

1. Color Formats

Hex: #3498db
RGB: (46, 204, 113)
HSL: (280, 60%, 50%)

2. Background Properties

The background shorthand property allows you to set multiple values at once, but it's important to know the individual components:

  • background-image: Sets an image as the background (url('image.jpg')).
  • background-repeat: Controls if an image repeats (no-repeat, repeat-x).
  • background-size: Adjusts image scaling (cover fills the area, contain fits inside).
  • background-position: Sets where the image starts (center, top right).

Study Tip: RGBA & Opacity

The 'A' in RGBA stands for Alpha (transparency). A value of 0.5 makes the color 50% transparent. This is better than the opacity property because it doesn't make the text inside the element transparent!

Quick Study Reference

  • Hexadecimal: 0-9 and A-F. #000000 is black, #FFFFFF is white.
  • Gradients: Created using linear-gradient() or radial-gradient(). They are technically images, not colors.
  • Attachment: background-attachment: fixed; creates a "parallax" scrolling effect.
