Understanding Variables

Variables are "containers" for storing information. In modern JavaScript, we use specific keywords to tell the browser how that container should behave.

Declaration Keywords

  • let: Used for variables that can change later (re-assignable).
  • const: Used for variables that cannot change (constant).
  • var: The older way to declare variables (mostly avoided in modern code).

Common Data Types

JavaScript variables can hold different types of data:

  • String: Text wrapped in quotes (e.g., "Hello").
  • Number: Decimals or integers (e.g., 25, 3.14).
  • Boolean: True or False values.
  • Array: A list of values (e.g., [1, 2, 3]).
