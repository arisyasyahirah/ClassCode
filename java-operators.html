Java Operators

Operators are used to perform operations on variables and values. In Java, operators are grouped into several categories based on their functionality.

Common Operator Types

  • Arithmetic Operators: Used to perform common mathematical operations (+, -, *, /, %).
  • Assignment Operators: Used to assign values to variables (=, +=, -=).
  • Comparison Operators: Used to compare two values (==, !=, >, <).
  • Logical Operators: Used to determine the logic between variables (&&, ||, !).

The Modulo Operator (%)

The modulo operator returns the division remainder. For example, 5 % 2 results in 1.

Example: Arithmetic
int sum = 100 + 50; System.out.println("Result: " + sum); // Output: Result: 150
