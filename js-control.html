Directing the Flow

Control flow is the order in which statements are executed. By default, code runs top-to-bottom, but we can change this using conditionals and loops.

1. Conditionals (If/Else)

Allows code to "decide" which path to take based on a true/false condition.

if (score > 50) {
  console.log("Pass");
} else {
  console.log("Fail");
}

2. Loops (For/While)

Repeats a block of code multiple times until a specific condition is met.

for (let i=0; i < 3; i++) {
  console.log("Repeat!");
}

Study Tip: The Switch Statement

When you have many "else if" conditions for the same variable, use a Switch statement. It's cleaner and faster for checking specific values.
