Interacting with the User

Dialog boxes are modal windows that pause the execution of code until the user interacts with them. JavaScript provides three primary types:

1. Alert

Used to display a simple message to the user. It only has an "OK" button.

alert("Hello World!");

2. Confirm

Asks a Yes/No question. Returns true if OK is clicked and false if Cancel is clicked.

confirm("Are you sure?");

3. Prompt

Asks the user to input text. Returns the string entered by the user, or null if they cancel.

prompt("Enter name:", "Guest");

Critical Note for Students

Remember that Prompt always returns data as a String. If you want to use the input for math, you must convert it using Number() or parseInt().
