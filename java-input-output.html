Java Input and Output

In Java, Output is used to show information to the user, while Input is used to get information from the user via the keyboard.

1. Java Output

The most common method to output data is System.out.println(), which prints text to the console and starts a new line.

  • print() - Prints text on the same line.
  • println() - Prints text and moves to the next line.

2. Java Input (Scanner)

To get user input, we use the Scanner class from the java.util package.

How to use Scanner:

  1. Import the class: import java.util.Scanner;
  2. Create an object: Scanner sc = new Scanner(System.in);
  3. Read input: sc.nextLine(); (for Strings) or sc.nextInt(); (for integers).
