Introduction to Java

Java is a popular, high level, class-based, object-oriented programming language. It was developed by Suns Microsystem and released in 1995.

Why is java so popular?

  • Platform Independent: Java code is compiled into bytecode, which can be run on any platform (Windows, Mac, Linux) that has a Java Virtual Machine (JVM).
  • Object-Oriented: It is built around the concept of objects, which helps in creating modular, reusable, and maintainable code.
  • Robust and Secure: It manages memory effectively and includes features like exception handling and garbage collection, making it highly reliable.
  • Vast Ecosystem: Used for enterprise-level systems, Android development, web applications, and scientific computing.

Java Basics : The WORA Principle

  • The "Write Once, Run Anywhere" principle is central to Java. When you write a Java program, the **Java Compiler** converts your source code (`.java` file) into **bytecode** (`.class` file).
  • This bytecode is then executed by the **Java Virtual Machine (JVM)**, which acts as an interpreter for the bytecode on the specific operating system.
  • This abstraction layer (the JVM) is what ensures your code runs the same way everywhere
