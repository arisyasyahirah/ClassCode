Java Loops

Loops are used to execute a block of code repeatedly as long as a specified condition is met. They are handy because they save time, reduce errors, and make code more readable.

Loop Logic Flowchart

\

Understanding how the computer decides to repeat code is easier with a visual flow:

Types of Java Loops

  • For Loop: Used when you know exactly how many times you want to loop through a block of code.
  • While Loop: Loops through a block of code as long as a specified condition is true.
  • Do/While Loop: Similar to the while loop, but it will execute the code block once before checking the condition.

The For Loop Syntax

The syntax consists of three parts: initialization, condition, and increment/decrement.

Example: Counting to 5
for (int i = 1; i <= 5; i++) { System.out.println("Iteration: " + i); }

Question: How many times will this loop run?

                
int x = 0;
while (x < 3) {
    System.out.println(x);
    x++;
}

                
                

                    Answer: 3 times. It will print 0, 1, and 2. When x becomes 3, the condition x < 3 becomes false.
