What is CSS?

Cascading Style Sheets (CSS) is the language we use to style an HTML document. While HTML provides the content (text, images, links), CSS provides the Presentation (colors, spacing, fonts, and layout).

The Three Types of CSS

There are three ways to add CSS to your website. Understanding these is essential for efficient development:

  • Inline CSS: Added directly to an HTML element using the style attribute.
    (Example: <h1 style="color:blue;">)
  • Internal CSS: Defined within a <style> tag inside the <head> of an HTML page.
  • External CSS: A separate .css file linked to the HTML. This is the best practice for professional websites.

CSS Syntax

CSS works by selecting an element and declaring how it should look. A CSS rule consists of a Selector and a Declaration Block.

The Cascade Effect

The "C" in CSS stands for Cascading. This means that if multiple rules apply to the same element, the browser follows a specific hierarchy to decide which rule "wins." Usually, the most specific rule or the one written last takes priority.

CSS Syntax Example
/* Selector { Property: Value; } */ h1 { color: darkblue; text-align: center; font-family: Arial; }
