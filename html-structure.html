HTML5 Structure & Syntax

Document Structure

Every HTML5 document follows a basic structure:

  • <!DOCTYPE html> - Declares the document type
  • <html> - Root element
  • <head> - Contains metadata
  • <body> - Contains visible content

HTML Syntax Rules

  • Tags are enclosed in angle brackets: <tag>
  • Most tags come in pairs: opening and closing
  • Tags must be properly nested
  • Attributes provide additional information
  • HTML5 is case-insensitive but lowercase is recommended

Proper Nesting

Tags must open and close in the correct order.

Block vs Inline Elements

  • Block: <p>, <div>, <h1>
  • Inline: <span>, <em>, <strong>
