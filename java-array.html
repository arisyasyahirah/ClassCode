Understanding Arrays

Arrays are used to store multiple values in a single variable, instead of declaring separate variables for each value. This makes it easier to manage collections of data, such as a list of student names or test scores.

Array Structure and Indexing

Arrays in Java are zero-indexed, meaning the first element starts at index 0. Understanding this is key to accessing the correct data.

Working with Arrays

  • Declaration: Use square brackets to define the type, like String[] or int[].
  • Size: Once an array is created, its size is fixed and cannot be changed.
  • Access: You use the index number inside brackets, e.g., myArray[0], to get or change a value.

Syntax Example

To declare an array of strings: String[] cars = {"Volvo", "BMW", "Ford"};

Example: Accessing an Array
String[] fruits = {"Apple", "Banana", "Cherry"}; System.out.println(fruits[0]); // This prints "Apple"
