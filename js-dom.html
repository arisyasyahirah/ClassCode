Data Structures & Page Manipulation

To build truly interactive websites, we must understand how JavaScript stores complex information in Objects and how it accesses HTML elements via the DOM.

Part 1: JavaScript Objects

Objects allow you to group related data (Properties) and functions (Methods) into one variable. This is known as Encapsulation.

let user = {
  name: "Alex",
  points: 100,
  login: function() { console.log("User Access Granted"); }
};

Part 2: The DOM Tree Hierarchy

The Document Object Model (DOM) treats the page as a tree of nodes. JavaScript climbs this tree to modify content instantly.

  • Selection: Use document.getElementById() or document.querySelector().
  • Modification: Change content via .innerHTML or .style.

Critical DOM Methods

  • .innerHTML - Changes the text/HTML inside an element.
  • .style.color - Changes the text color.
  • .style.backgroundColor - Changes the background color.
