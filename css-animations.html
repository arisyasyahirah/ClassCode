Modern UIs often use global "listeners" to handle broad commands. By listening for the Enter key, we can trigger a batch operation across all active DOM elements.

Why use transform?

Animating top and left causes the browser to "reflow" the layout. Using transform: translate() tells the computer to use the Graphics Card (GPU), making it much faster.
