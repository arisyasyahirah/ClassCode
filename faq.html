Java Basics

The JDK (Java Development Kit) is a superset of the JRE (Java Runtime Environment) and includes tools for developing Java applications. The JRE is a subset of the JDK and contains the Java Virtual Machine (JVM) and libraries needed to run Java applications. The JVM (Java Virtual Machine) is an abstract machine that provides a runtime environment for executing Java bytecode.

The main principles of OOP in Java are encapsulation, inheritance, polymorphism, and abstraction. Encapsulation involves bundling data and methods together and controlling access to them. Inheritance allows a class to inherit properties and methods from another class. Polymorphism enables objects of different types to be treated as instances of the same type through a common interface. Abstraction focuses on hiding complex implementation details and showing only essential features of an object.

The == operator compares the references of two objects, checking if they point to the same memory location. The .equals() method compares the actual content or value of two objects. For primitive types, == checks for equality of values, while for objects, it checks for reference equality unless overridden by a custom equals() method.

HTML Basics

Semantic HTML uses meaningful tags to describe the purpose of content, such as <header>, <nav>, <main>, <article>, <section>, and <footer>. This improves accessibility, SEO, and code readability. Screen readers use semantic tags to navigate content more effectively, and search engines can better understand the structure and context of your page.

The DOCTYPE declaration tells the browser which version of HTML the page is written in. For HTML5, use <!DOCTYPE html>. It must be the first line in your HTML document (before the <html> tag). This ensures the browser renders the page in standards mode rather than quirks mode.

div is a block-level element used to group and structure content, while span is an inline element used for styling or applying CSS to a portion of text. Use <div> for larger sections of content and <span> for smaller, inline elements.

CSS Basics

Margin is the space outside an element's border, while padding is the space inside the border. Margin controls spacing between elements, and padding controls spacing between the content and the border.

The alt attribute provides alternative text for images, which is crucial for accessibility. Screen readers use alt text to describe images to visually impaired users. It also displays if the image fails to load. Always include descriptive alt text that conveys the image's purpose or content.

JavaScript Basics

let and const are block-scoped variables introduced in ES6, while var is function-scoped. let allows reassignment of values, const declares a constant that cannot be reassigned, and var can be reassigned and hoisted. Use let for variables that will change and const for those that won't.

The == operator performs type coercion, comparing values after converting them to the same type. The === operator is strict equality, comparing both value and type without coercion. Use === for more predictable comparisons and to avoid unexpected behavior from automatic type conversion.

JavaScript has several data types: string (text), number (integers and decimals), boolean (true or false), null (intentional absence of a value), undefined (a variable that has not been assigned a value), symbol (unique identifier), and object. Additionally, there is a special type called bigint, which is used for large integers.

Yes, HTML is the foundation of web development. HTML provides the structure, CSS adds styling, and JavaScript adds interactivity. You should learn HTML first, then CSS, and finally JavaScript. Understanding HTML is essential before moving to other web technologies.

SQL Basics

INNER JOIN returns only rows that have matching values in both tables. LEFT JOIN returns all rows from the left table and matching rows from the right table. RIGHT JOIN returns all rows from the right table and matching rows from the left table.

SQL indexes are database objects that improve the speed of data retrieval operations on a table. They work similarly to an index in a book, allowing the database engine to quickly locate and access specific rows without scanning the entire table. Indexes are particularly useful for large tables and frequently queried columns.
