HTML5 Practice & Exercises

This page contains practical exercises, examples, and challenges to test your HTML5 knowledge. Try solving these problems to reinforce your learning.

Beginner Exercises

Exercise 1: Basic Page Structure

Task: Create a complete HTML5 document with the following:

  • Proper DOCTYPE declaration
  • HTML element with lang attribute
  • Head section with title and charset
  • Body with a heading, paragraph, and horizontal rule
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>My First Page</title>
</head>
<body>
    <h1>Welcome to HTML</h1>
    <p>This is my first HTML page.</p>
    <hr>
</body>
</html>

Exercise 2: Text Formatting

Task: Create a paragraph with the following formatting:

  • Bold important terms
  • Italicize book titles
  • Mark important text
  • Show chemical formula: H2O
  • Show mathematical equation: x2 + y2 = z2
<p>
    <strong>HTML</strong> is the standard markup language for web pages.
    I'm reading <em>Web Development for Beginners</em>.
    Remember: <mark>always validate your code</mark>.
    Water formula: H<sub>2</sub>O
    Pythagorean theorem: x<sup>2</sup> + y<sup>2</sup> = z<sup>2</sup>
</p>

Intermediate Exercises

Exercise 3: Navigation Menu

Task: Create a navigation menu with:

  • Unordered list with 5 items
  • Each item should be a link
  • One link should open in new tab
  • Add a logo image before the menu
<nav>
    <img src="logo.png" alt="Website Logo" width="50">
    <ul>
        <li><a href="index.html">Home</a></li>
        <li><a href="about.html">About</a></li>
        <li><a href="services.html">Services</a></li>
        <li><a href="blog.html">Blog</a></li>
        <li><a href="https://external.com" target="_blank">External</a></li>
    </ul>
</nav>

Exercise 4: Registration Form

Task: Create a registration form with:

  • Name (text input, required)
  • Email (email input, required)
  • Password (password input, min 8 chars)
  • Date of birth (date input)
  • Country (dropdown with 3 options)
  • Subscribe checkbox
  • Submit button
<form>
    <label for="name">Full Name:</label>
    <input type="text" id="name" name="name" required>
    
    <label for="email">Email:</label>
    <input type="email" id="email" name="email" required>
    
    <label for="password">Password:</label>
    <input type="password" id="password" name="password" minlength="8" required>
    
    <label for="dob">Date of Birth:</label>
    <input type="date" id="dob" name="dob">
    
    <label for="country">Country:</label>
    <select id="country" name="country">
        <option value="malaysia">Malaysia</option>
        <option value="singapore">Singapore</option>
        <option value="indonesia">Indonesia</option>
    </select>
    
    <label>
        <input type="checkbox" name="subscribe"> Subscribe to newsletter
    </label>
    
    <button type="submit">Register</button>
</form>

Exercise 6: Complete Blog Layout

Task: Create a complete blog layout with:

  • Semantic HTML structure
  • Header with logo and navigation
  • Main content area with articles
  • Sidebar with recent posts and categories
  • Footer with contact info
  • Responsive design
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Blog</title>
    <style>
        * { box-sizing: border-box; margin: 0; padding: 0; }
        body { font-family: Arial, sans-serif; line-height: 1.6; }

        nav ul { display: flex; list-style: none; gap: 1rem; }
        nav a { color: white; text-decoration: none; }

            display: grid; 
            grid-template-columns: 3fr 1fr; 
            gap: 2rem; 
            padding: 2rem; 
            max-width: 1200px; 
            margin: 0 auto;

            background: #333; 
            color: white; 
            text-align: center;

        }

</head>
