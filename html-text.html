HTML Text Formatting & Inline Elements

Inline elements are used within block-level elements to format content without creating new lines. They only take up as much width as necessary.

Text Formatting Elements

Basic Formatting

Common Text Formatting Tags:

Element Purpose Example
<strong> Important text (semantic) This is important
<b> Bold text (visual only) Bold text
<em> Emphasized text (semantic) Emphasized text
<i> Italic text (visual only) Italic text
<mark> Highlighted text Highlighted text
<small> Smaller text Small text
<del> Deleted text Deleted text
<ins> Inserted text Inserted text
<sub> Subscript H2O
<sup> Superscript x2

Semantic vs Non-semantic Formatting

Semantic elements convey meaning about the content's importance:

Warning: The system will shut down in 5 minutes.

Please read the instructions carefully.

Non-semantic elements are purely visual:

This text is bold and this text is italicized.

Inline Text Elements

Other Useful Inline Elements:

Element Purpose Example
<span> Generic inline container Styled text
<code> Inline code Use console.log() for debugging
<kbd> Keyboard input Press Ctrl + C to copy
<samp> Sample output Error: File not found
<var> Variable The variable x represents...
<abbr> Abbreviation HTML
<cite> Citation The Art of Programming
<q> Short quotation To be or not to be

Best Practices

  • Use <strong> for important text instead of <b>
  • Use <em> for emphasis instead of <i>
  • Always provide title attribute for <abbr> elements
  • Use <span> when no semantic element is appropriate
  • Combine inline elements for complex formatting
