Modern Layout Techniques

Gone are the days of using tables or floats for layout. Today, we use Flexbox for alignment in one direction and CSS Grid for full-page structures.

1. CSS Flexbox (1D Layout)

Flexbox is best for aligning items in a row or a column. It is highly flexible (hence the name) and makes centering elements incredibly easy.

  • flex-direction: Row (horizontal) or Column (vertical).
  • justify-content: Aligns items along the Main Axis (left, center, space-between).
  • align-items: Aligns items along the Cross Axis (top, center, bottom).

2. CSS Grid (2D Layout)

Grid is designed for two-dimensional layoutsâ€”both rows and columns at the same time. It is perfect for complex dashboard-style layouts.

  • grid-template-columns: Defines how many columns and their width (e.g., 1fr 1fr 1fr).
  • gap: The spacing between rows and columns.

Which one should I use?

Use Flexbox when: You have a small group of items to align (like a navbar or buttons inside a card).

Use Grid when: You are building the overall structure of the whole web page.

Study Tips for Success

  • The Golden Rule: To use these properties, you MUST apply display: flex or display: grid to the Parent element.
  • Centering Hack: To perfectly center anything, use:
    display: flex; justify-content: center; align-items: center;
  • The FR Unit: In Grid, 1fr stands for "fractional unit." Using grid-template-columns: 1fr 2fr; makes the second column twice as wide as the first.
