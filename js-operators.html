The Logic of JavaScript

Operators allow us to manipulate data, perform math, and make decisions in our code. They are divided into several key categories every developer must master.

1. Arithmetic Operators

  • % (Modulus): Returns the remainder (e.g., 10 % 3 = 1). Great for finding even/odd numbers.
  • ++ / -- : Increment or Decrement a value by 1.
  • ** (Exponentiation): ES6 power operator (e.g., 5 ** 2 = 25).

2. Comparison Operators

  • == vs === : == checks value only; === checks value AND data type (Strict Equality).
  • != vs !== : Not equal and Strict not equal.
  • >= / <= : Greater than or equal / Less than or equal.

3. Logical Operators (Decision Making)

Used to combine multiple boolean conditions:

  • && (AND): True only if BOTH sides are true.
  • || (OR): True if at least ONE side is true.
  • ! (NOT): Reverses the result (True becomes False).

4. Operator Precedence (PEMDAS)

Just like math, JavaScript follows an order: Parentheses () first, then Multiplication/Division, then Addition/Subtraction.
