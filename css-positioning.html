Beyond the Normal Flow

Every element in HTML follows a Visual Formatting Model. By default, they are static, meaning they follow the natural scroll of the page. To create modern interfaces like sticky headers or pop-up modals, we must break this flow.

2. Mastery of Z-Index & Stacking Context

Overlap is managed by the Z-Index. However, many students get frustrated when a higher Z-index doesn't work. This is usually due to Stacking Contexts.

Rule of Thumb: If an element has a parent with z-index: 1, no matter how high the child's z-index is (even 9999), it will still be trapped behind any element that has a parent with z-index: 2.

Exam Strategy

  • Absolute needs a Parent: Always check if the parent has position: relative. If not, your element will fly to the top of the page.
  • The Coordinate Trap: Remember that bottom: 0 means "stick to the bottom edge," not "move down."

Pro Trick

To center an element that is position: absolute, use this formula:

top: 50%; left: 50%;
transform: translate(-50%, -50%);
