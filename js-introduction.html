Introduction to JavaScript

JavaScript is a popular, object-based, client-side scripting language. It was originally developed by Brendan Eich at Netscape and is now the standard for creating interactive web pages.

What is JavaScript?

  • Client-Side: It runs directly in the user's web browser, not on the server.
  • Interpreted: JavaScript code is interpreted "on the fly" by the browser's engine (like Chrome's V8) without needing a separate compilation step.
  • Case-Sensitive: Unlike HTML, JavaScript treats uppercase and lowercase letters differently (e.g., myVar is not myvar).
  • Interactive: It allows pages to respond to user events like clicks, mouse movements, and form submissions.

The Three Layers of Web Development

To build a modern dynamic website, you need three main components working together:

  • HTML (Structure): The content and bones of the page.
  • CSS (Presentation): The style, colors, and layout.
  • JavaScript (Behavior): The interactivity and logic.

Your First Script: Internal JavaScript

JavaScript is usually placed within <script> tags inside an HTML document, often just before the closing </body> tag for better performance.

Example
<script> // This function creates a simple alert box function sayHello() { alert("Welcome to ClassCode JavaScript Tutorial!"); } sayHello(); </script>

Question: Is JavaScript the same as Java?

Answer: No. Despite the similar names, they are completely different languages. Java is a compiled, class-based language, while JavaScript is a lightweight, interpreted scripting language designed for the web.
